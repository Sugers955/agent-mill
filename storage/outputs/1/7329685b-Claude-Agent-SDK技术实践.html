What is it

什么是
Claude Agent SDK

Anthropic 官方提供的智能体开发框架，让你可以用代码精确控制 Claude 的行为、工具和响应流程。

不同于简单的 API 调用，Agent SDK 提供了结构化对话流、多工具编排、状态管理等企业级能力。它让 Claude 从"聊天机器人"升级为"可编程智能体"。

"Agent SDK 不是让 AI 更聪明，
而是让 AI 的行为更可预测、可控制。"

Act I / 05 · 07 / 20
Code Example

基础 Agent 定义

  • @function_tool 装饰器注册工具
  • Agent 类封装 system prompt
  • Runner.run() 启动对话循环
  • 支持异步流式响应
关键：工具函数必须是类型标注的，SDK 会自动生成 JSON Schema 
from agents import Agent, Runner, function_tool

@function_tool
async def get_weather(
    location: str,
    unit: str = "celsius"
) -> str:
    """Get weather for a location."""
    return f"Weather in {location}: 24°{unit}"

# Define the agent
agent = Agent(
    name="WeatherBot",
    instructions="You are a helpful weather assistant.",
    tools=[get_weather]
)

# Run
result = await Runner.run(
    agent,
    "What's the weather in Tokyo?"
)
print(result.final_output)

Layer 1

认证与鉴权

确保只有合法用户能访问 Agent，每个请求都可追溯到具体身份。

永远不要将 Anthropic API Key 暴露给前端浏览器

Layer 4

输出内容审查

Claude 的响应同样需要审查，防止敏感信息泄露和不当内容输出。

PII 检测
身份证号、银行卡、手机号等敏感信息自动脱敏
内容审核
暴力、仇恨、成人内容分级过滤
合规检查
行业法规、公司政策、地域法律合规
输出审查流水线
1. Claude 生成响应
2. 敏感信息扫描
3. 内容安全分类
4. 合规规则匹配
5. 脱敏处理/放行
