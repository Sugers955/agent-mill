## 模块架构

### 后端模块

```
backend/app/
├── api/                    # API 路由层
│   ├── admin.py           # 管理接口
│   ├── chat.py            # 对话接口
│   └── command_center.py  # 指挥中心接口
├── auth/                   # 认证模块
│   ├── jwt.py             # JWT 处理
│   ├── sso.py             # SSO/LDAP
│   └── permissions.py     # 权限控制
├── core/                   # 核心配置
│   ├── config.py          # 配置管理
│   ├── security.py        # 安全工具
│   └── encryption.py      # 加密工具
├── db/                     # 数据层
│   ├── models.py          # SQLAlchemy 模型
│   ├── init_db.py         # 数据库迁移
│   └── seed.py            # 种子数据
├── runtime/                # Agent 运行时
│   ├── agent_runner.py    # Agent 执行器
│   ├── pack_engine.py     # 方案包引擎
│   └── workflow_engine.py # 工作流引擎
├── services/               # 业务服务
│   ├── rag.py             # RAG 服务
│   ├── memory.py          # 记忆系统
│   ├── learning.py        # 自学习
│   └── alert.py           # 告警服务
└── main.py                 # 应用入口
```

## 数据流

### 对话流式响应

```
用户输入
    │
    ▼
┌─────────────────┐
│   API Gateway   │
│   (认证/限流)   │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   Chat Service  │
│   (创建对话)    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  Agent Runner   │
│  (执行 Agent)   │
└────────┬────────┘
         │
    ┌────┴────┐
    │         │
    ▼         ▼
┌────────┐ ┌────────┐
│Anthropic│ │OpenAI  │
│ SDK    │ │兼容路径│
└────┬───┘ └────┬───┘
     │          │
     └────┬─────┘
          │
          ▼
┌─────────────────┐
│   SSE Stream    │
│   (事件推送)    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  客户端渲染     │
│  (Markdown/工具)│
└─────────────────┘
```

### 技能执行流

```
Agent 调用技能
    │
    ▼
┌─────────────────┐
│  Skill Router   │
│  (判断类型)     │
└────────┬────────┘
         │
    ┌────┴────┐
    │         │
    ▼         ▼
┌────────┐ ┌────────┐
│ path   │ │callable│
│  型    │ │  型    │
└────┬───┘ └────┬───┘
     │          │
     ▼          ▼
┌────────┐ ┌────────┐
│ ZIP    │ │Python  │
│ 解压   │ │ import │
│ 执行   │ │ 调用   │
└────┬───┘ └────┬───┘
     │          │
     └────┬─────┘
          │
          ▼
┌─────────────────┐
│   返回结果      │
└─────────────────┘
```

```
用户消息
    │
    ▼
┌─────────────────┐

└────────┬────────┘
         │
         ▼

         │

         │
         ▼

└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  Agent 处理     │
│  (结合上下文)   │

```

## 安全架构

### 七层防护

```
┌─────────────────────────────────────────┐
│  Layer 7: API Key 加密（Fernet）         │
├─────────────────────────────────────────┤
│  Layer 6: 下载令牌（一次性）             │
├─────────────────────────────────────────┤
│  Layer 5: 文件沙箱（cwd 限制）           │
├─────────────────────────────────────────┤
│  Layer 4: 技能扫描（AST 静态分析）       │
├─────────────────────────────────────────┤
│  Layer 3: 输入过滤（12 条正则）          │
├─────────────────────────────────────────┤
│  Layer 2: 安全前缀（系统提示词）         │
├─────────────────────────────────────────┤
│  Layer 1: 工具白名单（运行时）           │
└─────────────────────────────────────────┘
```

### 认证流程

```
用户登录
    │
    ▼
┌─────────────────┐
│  验证凭据       │
│  (密码/SSO/LDAP)│
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  生成 Token     │
│  (JWT 双 token) │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  返回客户端     │
│  (access+refresh)│
└────────┬────────┘
         │
    ┌────┴────┐
    │         │
    ▼         ▼
┌────────┐ ┌────────┐
│API 请求│ │Token   │
│        │ │刷新    │
└────┬───┘ └────┬───┘
     │          │
     ▼          ▼
┌────────┐ ┌────────┐
│验证    │ │生成新  │
│过期    │ │Token   │
└────┬───┘ └────┬───┘
     │          │
     └────┬─────┘
          │
          ▼
┌─────────────────┐
│  继续请求       │
└─────────────────┘
```

## 扩展点

### 添加新的 LLM 供应商

1. 在 `agent_runner.py` 中添加 provider 判断
2. 实现对应的 API 调用逻辑
3. 在 `models.py` 中注册新的 provider 类型
4. 更新前端模型配置页面

### 添加新的技能类型

1. 在 `skill_executor.py` 中添加类型判断
2. 实现对应的执行逻辑
3. 在 `models.py` 中注册新的 skill 类型
4. 更新前端技能管理页面

### 添加新的 MCP 传输协议

1. 在 `mcp_client.py` 中添加协议支持
2. 实现对应的连接和通信逻辑
3. 在 `models.py` 中注册新的协议类型
4. 更新前端连接器配置页面
